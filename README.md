# Monthly Mortgage Calculator

## Overview
This project helps users calculate their monthly payments for fixed-rate mortgages based on principal, interest rate, and loan term. A brief explanation of mortgages is provided. Users can then navigate to the interactive form for calculation. 
The webpage incorporates Monopoly-themed visual elements for more user engagement. Information on rules and tips in relation to the board game is provided to the right of the interactive form.

## Technologies Used
- HTML
- CSS
- JavaScript
